# Wibble Client Core

![GitHub License](https://img.shields.io/github/license/windlasstech/wibble-client-core)
[![GitHub issues](https://img.shields.io/badge/issue_tracking-GitHub-blue.svg)](https://github.com/yunseo-kim/agent-toolbox/issues)

The wibble-client-core provides the client-side security and protocol core for the Wibble ecosystem. It focuses on MLS/OpenMLS-based security primitives, message canonicalization, and cryptographic hashing utilities.

## Architecture Overview

- **Core modules**: Security core, protocol logic, and shared utilities
- **Responsibility**: MLS/OpenMLS-based security primitives, canonicalization, and hashing
- **Design**: Protocol-agnostic core with platform bindings
- **Note**: Tech stack to be determined per protocol-spec requirements
